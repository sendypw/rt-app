# Fitur: Dasbor (Halaman Utama)

Dasbor adalah halaman pertama yang Anda lihat setelah berhasil login. Halaman ini dirancang untuk memberikan ringkasan cepat dan akses ke tindakan terpenting terkait tugas siskamling Anda.

## Komponen Utama

1.  **Kartu Tugas Berikutnya:**
    *   **Fungsi:** Menampilkan tanggal tugas siskamling Anda yang paling dekat di masa depan.
    *   **Tampilan:** Menampilkan tanggal besar dan nama bulan untuk visibilitas yang jelas.
    *   **Tombol "Check-in":** Jika jadwal tugas Anda adalah **hari ini**, sebuah tombol **"Check-in Sekarang"** akan muncul. Klik tombol ini untuk mengonfirmasi kehadiran Anda pada hari itu. Setelah di-klik, status Anda akan diperbarui dan tombol akan berubah menjadi label "Sudah Check-in".

2.  **Kartu Laporan Tugas:**
    *   **Fungsi:** Memungkinkan Anda untuk mengirimkan laporan setelah tugas siskamling selesai.
    *   **Ketersediaan:** Tombol **"Kirim Laporan"** hanya akan aktif jika Anda sudah melakukan *check-in* pada hari tugas Anda.
    *   **Alur Pelaporan:**
        1.  Klik tombol **"Kirim Laporan"** (atau "Lihat Laporan" jika sudah pernah mengirim).
        2.  Sebuah jendela dialog akan muncul.
        3.  Tulis ringkasan kejadian atau observasi selama Anda bertugas di area teks yang tersedia.
        4.  (Opsional) Anda dapat melampirkan foto sebagai bukti atau dokumentasi dengan memilih tab "Unggah File" atau "Ambil Foto".
        5.  Klik **"Kirim"** untuk menyimpan laporan Anda.
