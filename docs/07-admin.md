# Fitur: Panel Admin

Panel Admin adalah halaman khusus yang hanya dapat diakses oleh pengguna dengan peran 'admin'. Halaman ini menyediakan alat untuk mengelola data inti aplikasi, terutama jadwal siskamling.

## Akses

*   Menu "Admin" akan muncul di sidebar kiri hanya jika Anda login sebagai pengguna dengan peran 'admin'.
*   Jika pengguna biasa mencoba mengakses halaman ini secara langsung, mereka akan otomatis dialihkan.

## Fungsionalitas Utama

Panel admin berpusat pada komponen **"Kelola Jadwal"**.

1.  **Menambah Tugas Baru:**
    *   Klik tombol **"Tambah Tugas"**.
    *   Sebuah jendela dialog akan muncul.
    *   Pilih **warga** dari daftar dropdown yang tersedia.
    *   Pilih **tanggal** tugas menggunakan pemilih kalender.
    *   Klik **"Simpan"**. Jadwal baru akan ditambahkan ke daftar.

2.  **Mengubah Tugas yang Ada:**
    *   Cari tugas yang ingin diubah dalam tabel.
    *   Klik ikon **pensil (Ubah)** di sebelah kanan baris tugas.
    *   Jendela dialog akan muncul, terisi dengan data tugas yang ada.
    *   Ubah warga atau tanggal sesuai kebutuhan.
    *   Klik **"Simpan"**. Perubahan akan langsung diterapkan.

3.  **Menghapus Tugas:**
    *   Cari tugas yang ingin dihapus.
    *   Klik ikon **tong sampah (Hapus)** di sebelah kanan baris tugas.
    *   Sebuah dialog konfirmasi akan muncul untuk memastikan Anda tidak salah klik.
    *   Klik **"Hapus"** untuk mengonfirmasi. Tugas tersebut akan dihapus secara permanen dari sistem.
