# **App Name**: Jaga RT App

## Core Features:

- Custom Login: Authenticate users based on house number and a static password.
- Role-Based Access Control (RBAC): Restrict access to certain features based on user role ('admin' or 'warga').
- Duty Schedule Display: Display the duty schedule fetched from Firestore.
- Firestore Integration: Store and retrieve duty schedule data from Firestore.
- Admin Panel: Allow admins to manage and update the duty schedule (CRUD operations in Firestore).
- Attendance Tracking: Record attendance of residents during their duty shifts.
- Report Submission: Allow residents to submit reports after completing their duty shifts.
- Schedule Swap Request: Allow residents to request a schedule swap with other residents.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and community.
- Background color: Light gray (#F0F0F0), providing a clean, neutral backdrop.
- Accent color: Orange (#FF9800) for highlighting important actions and information.
- Body and headline font: 'PT Sans' sans-serif for readability and modern look.
- Use simple, clear icons to represent different duty shifts and actions.
- Responsive layout that adapts to different screen sizes, ensuring accessibility on various devices.
